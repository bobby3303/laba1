# Задание 1

## Задача 1

### Текст задачи

Сформировать список из вводимых строк.

### Алгоритм решения

Считываем с клавиатуры строки в функции BuildList. Если строка не пустая - вызываем функцию снова чтобы получить конец списка и добавляем текущую строку в начало. Если пустая, значит ввод закончен - возвращаем пустой список. Выводим список

# Задание 2

## Задача 1

### Текст задачи

Найти суммарную длину введенных строк. (Признак конца ввода — пустая строка)

### Алгоритм решения

Используем рекурсивную функцию для ввода строк. На спуске прибавляем длину введенной строки к результату следующих шагов рекурсии. Конец рекурсии - пустая строка. На подъёме возвращаются значения длин строк и складываются. выводим резудьтат

# Задание 3

## Задача 1

### Текст задачи

Опишите функции, осуществляющие основные математические операции над комплексными
числами (сложение, вычитание, умножение, деление, возведение в степень). Использовать
готовый класс Complex запрещено.

### Алгоритм решения

Вводим операции и значения чисел, вводя отдельно действительные и мнимые части. В сложении и вычитании складываем и вычитаем соответствующие части. В умножении согласно алгоритму умножения, умножаем части исходных чисел крест на крест чтобы получить действительные и мнимые части. В делении сначала вычисляем знаменатель, при умножении дроби на сопряженное с делителем число. Дальше домножаем числитель на то же самое сопряженное число и делим на полученный раннее знаменатель чтобы получить действительные и мнимые части. В возведении в степень рекурсивно умножаем текущее число на результат работы функции от этого же числа и степени - 1. рекурсивно спускаемся пока степень не будет <=0.
